Investigation Overview¶

I am interested in what affects the borrower’s APR nad i'm using these feature to explore that Term, BorrowerAPR, BorrowerRate, LoanOriginalAmount, LoanStatus, ProsperRating (Alpha), ProsperRating (numeric), MonthlyLoanPayment, EmploymentStatus

Dataset Overview¶

The data set contains 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others.
